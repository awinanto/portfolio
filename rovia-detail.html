Rovia — AI-Powered 3D Virtual Assistant

Case Study: Rovia AI

Project Vision

Developed at Inker Robotics, Rovia is a functional 3D virtual assistant designed for lifelike visual presence and real-time voice interaction . The project replaces standard chatbot interfaces with a digital human capable of natural speech and emotion.

  • MetaHuman Creator: Used for detailed character modeling and a fully rigged skeleton.
  • Quixel Bridge: Facilitated the seamless import of MetaHuman assets into the engine.
  • Quixel Fab: Provided the assets and plugins used to assemble the custom virtual room.
  • Unreal Engine 5: The primary platform for real-time rendering, lighting, and performance.

Technologies Used

Technical Workflow

I adopted a modular approach to bridge 3D animation with AI logic within Unreal Engine 5:

  • Character Setup: Configured animation blueprints and facial control rigs for real-time interaction.
  • Convai Integration: Linked the Convai plugin via API keys to handle voice interaction and knowledge base responses.
  • Visual Sync: Mapped lip-syncing and facial expressions directly to voice output for realism.
  • Packaging: Compiled the final project into a standalone Windows .exe application.

Technical Challenges

I optimized environment props and lighting to resolve performance drops caused by heavy assets. This ensured a smooth experience in the final "Shipping" configuration of the standalone application.
